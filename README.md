# Project Test Final

Este projeto demonstra TDD e BDD em JavaScript, com uma aplicação web integrada.

## Instalação
1. Instale Node.js.
2. Execute `npm install` na raiz do projeto.

## Execução
- Testes TDD: `npm test`
- Testes BDD: `npm run cucumber`
- Web App: Abra `public/index.html` no navegador.

## Estrutura
- `src/tdd/`: Módulo TDD (validações).
- `src/bdd/`: Módulo BDD (autenticação e carrinho).
- `public/`: Interface web.
